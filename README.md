# Thesis

This repository contains the thesis report and presentation materials. It is kept private because the public code repository is sufficient for readers who want to inspect the formal Lean development behind the thesis.

The public code repository is here:

- [Bachelor-Thesis-Code](https://github.com/hato1883/Bachelor-Thesis-Code)

## Layout

- `report/` contains the thesis source, appendix material, figures, and build files for the written report.
- `presentation/` contains the slide deck sources and build files for the talk.
- `lean-code/` contains the Lean project used to formalize the semantics and proofs.

## Notes

The main thesis PDF is produced from the report sources. The presentation is maintained separately so it can evolve independently from the written thesis.
